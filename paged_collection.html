Paged Collection¶

class braintree.paged_collection.PagedCollection(query, collection, klass)¶

A class representing a page of search results.

current_page_size¶
Returns the number of items on this page.
is_last_page¶
Returns whether this is the last page of results.
next_page()¶
Returns another PagedCollection representing the next page of results or None if this is the last page.
total_pages¶
Returns the total number of pages of search results.
