Transparent Redirect¶

class braintree.transparent_redirect.TransparentRedirect¶

A class used for Transparent Redirect operations

static confirm(query_string)¶

Confirms a transparent redirect request. It expects the query string from the redirect request. The query string should _not_ include the leading “?” character.

result = braintree.TransparentRedirect.confirm("foo=bar&id=12345")
static url()¶
Returns the url for POSTing Transparent Redirect HTML forms
