Resource Collection¶

class braintree.resource_collection.ResourceCollection(query, results, method)¶

A class representing results from a search. Iterate over the results by calling items:

results = braintree.Transaction.search("411111")
for transaction in results.items:
    print transaction.id
first¶
Returns the first item in the results.
items¶
Returns a generator allowing iteration over all of the results.
maximum_size¶
Returns the approximate size of the results. The size is approximate due to race conditions when pulling back results. Due to its inexact nature, maximum_size should be avoided.
